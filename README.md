# URL Spinner

## Credits

[tomnomnom/waybackurls](https://github.com/tomnomnom/waybackurls)

## Description

A port of waybackurls using `rust`.

## Usage

```sh
urlspinner "domain"
```

or via pipe pattern:

```sh
echo "domain" | urlspinner
```

The other configuration files, follow the base of `waybackurls`.
